# Telegram Notifyer


## Description

Этот небольшой проект разработан специально для запуска на микро хостинге, требуемый объем диска менее 10 Мб. Он позволяет развернуть бот для оповещения через Телеграм о поступлении новых заказов у продавца на OZON площадке. А так же мониторить статус "доступен"/"недоступен" для своих веб-сайтов.

Оповещение бот отправляет в привязанные групповые чаты. Привязка чатов происходит при добавлении бота как участника в группу.


## Installing

1. Скопировать весь проект в выбранную папку на веб-сайте.
2. Исполнить команду:
    ```bash
    $ composer update --no-dev
    ```
3. Скопировать файл **.htaccess.example** в **.htaccess** и отредактировать:
    - BOT_JOBS  *список заданий*
    - ROOT_URI  *папка к которой мы планируем обращаться через веб*
    - SRC_URI  *папка в которой находится проект (обычно = ROOT_URI)*
    - BOT_INTERNAL_ID  *номер бота в нашей базе*
    - BOT_HOST_ID  *номер хостинга (для мониторинга своих веб-сайтов)*
    - BOT_TOKEN  *API токен телеграм бота*
    - ADMIN_CHAT_ID  *API id админского чата*
    - CRYPTO_KEY  *1й набор символов для ключа шифрования*
    - CRYPTO_SALT  *2й набор символов для ключа шифрования*
    - RewriteBase  *должно быть = ROOT_URI*
4. В корне на хостинге создать файл **ini_h.php**
   ```php
    <?php
    define('DB_HOST', 'хостинг базы данных');
    define('DB_USER', 'имя пользователя базы данных');
    define('DB_PASS', 'пароль');
    define('DB_DB', 'имя базы');
   ```
5. Используя файл *./TNotifyer/Database/tables.sql* создать структуру базы данных.
6. Запустить на сайте */set-webhook*


## Configuration

1. Предоставить боту доступ до OZON API используя команды из админского чата:
    ```
    /ozonsetid
    <CLIENT ID>
    /ozonsetkey
    <API KEY>
    ```
2. Создать на хостинге в планировщике задачу запускаемую каждую минуту:
    ```bash
    wget -O /dev/null "<путь до сайта>/run-jobs"
    ```


## Telegram Bot Commands

Команды доступные в Телеграм из админского чата.

1. **/help** - отобразить список комманд
2. **/info** - информация о приложении на хостинге
3. **/jobs** - отобразить список заданий
4. **/mainchats** - список привязанных чатов
5. **/test** - отправить тестовое сообщение в привязанные чаты
6. **/ozon** - информация об OZON аккаунте
7. **/ozonsetid** - установить OZON CLIENT ID
8. **/ozonsetkey** - установить OZON API KEY


## Site Commands

Команды доступные на сайте.

1. **/run-jobs** - исполнение списка заданий
2. **/ozon** - проверка новых заказов на OZON
3. **/ozon-status** - проверка статусов заказов на OZON
4. **/ping-websites** - проверка статуса веб-сайтов
5. **/watch-dog** - проверка активности бота
6. **/set-webhook** - привязка вебхука для бота
7. **/remove-webhook** - удаление привязки вебхука
8. **/up** - проверка API сообщений для бота (если нет привязанного вебхука)
9. **/info** - информация о приложении
10. **/bot-info** - информация о телеграм боте
11. **/ozon-info** - информация об OZON аккаунте
12. **/log** - отображение последних событий в приложении
13. **/last** - отображение последних заказов
14. **/status** - отображение статуса заказов
15. **/updates** - отображение последних API сообщений для бота
16. **/test-msg** - отправка тестового сообщения в привязанные чаты
17. **/ozon-test** - тестовое обращение к списку заказов на OZON
18. **/crypto-test** - запуск теста для библиотеки шифрования
